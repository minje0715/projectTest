#프로젝트 기획안  
작성일 : 2022. 05. 06

작성자 : 김민제

#####프로젝트 명:<span style="color:red"> 쇼핑몰을 구현해보자!


**기획 의도: ** 
>쇼핑몰에서 상품을 구매하기 위한 프로그램입니다  
  
>일반적인 쇼핑몰 사이트들이 어떤식으로 프로그래밍 되어 구현되는지 궁금했고 직접 구현해보면 다른 사이트들도 어떻게 구현되는지 알기 쉽게 접근할 수 있지 않을까 하는 생각을 했습니다.  

>회원가입 및 로그인을 할 수 있고 포인트를 충전하여 상품을 구매 할 수 있습니다.

**벤치마킹: ** 
>네이버 쇼핑몰에 들어가서 메인홈페이지 구조를 확인하였고 구글링을 통해 구현을 어떻게 하는지 구조를 찾아봤습니다. 로그인 및 회원가입을 봤고 장바구니 네이버페이 로 구매하는부분 , 포인트로 구매하는부분등을 참고하여 만들었습니다.  

**주요 기능: **
> 1.회원가입, 2.로그인, 3.내정보관리, 4.상품리스트, 5.상품주문, 6.종료  
손님을 위한 프로그램이므로 관리자가 할 수 있는부분은 없습니다.  
1.회원가입을 하면 아이디 비밀번호 이메일 핸드폰번호 성별 을 입력 할 수 있고 중복된 아이디가 있으면 중복된 아이디가 있다고 나오며 회원가입에 실패합니다.  
2.아이디 비밀번호를 쳐서 로그인을 할 수 있고 회원가입되지 않은 아이디 비밀번호를 입력 할 시에 아이디 비밀번호를 확인해달라는 문구가 나옵니다  
3.내정보관리에 들어가면 나의 정보 포인트충전 회원탈퇴 를 할 수 있으며
비회원으로 쇼핑몰을 이용 할 수 있기 때문에 내정보관리로 들어가서 목록을 이용할시에는 모두 아이디와 비밀번호를 필요로합니다
포인트 충전에는 최솟값, 최댓값 제약을 걸어놨습니다  
4.상품리스트에는 관리자가 미리 올려놓은 상품들을 구매 하고 목록을 확인 할 수 있습니다.  
5.주문을 하려면 로그인을 하여 회원임을 인증하고 포인트를 충전해놓은 상태에서 상품을 주문할 수 있습니다. 실수로 주문 버튼을 누를 수도 있기 때문에 한번더 확인창이 뜨고 내가 확인한 상태에서 주문을 할 수 있게 구현해놨습니다  
6.종료버튼을 누르면 프로그램이 종료됩니다.
